# Sample Dashboard Demo

Demonstrates KACA's web application generation capability.

## Quick Start

From repository root:

```bash
python examples/sample-dashboard/run_demo.py
```

Open `examples/sample-dashboard/output/dashboard.html` in your browser.

## What This Demonstrates

- `KDSTheme`: Brand-compliant styling with Kearney Purple (#7823DC)
- `KDSData`: CSV data loading with schema validation
- `KDSWebApp`: Static HTML generation with:
  - KPI metric cards
  - Interactive Plotly charts (bar, line)
  - Filterable data tables
  - Responsive 2-column layout
  - Embedded data (no server needed)
  - Dark mode styling

## Files

| File | Purpose |
|------|---------|
| `spec.yaml` | Dashboard specification (KACA format) |
| `data/sales.csv` | Sample sales data |
| `run_demo.py` | Generation script |
| `output/dashboard.html` | Generated output (after running demo) |

## Sample Data

The demo uses synthetic sales data with:
- 4 regions (North, South, East, West)
- 2 products (Widget A, Widget B)
- 2 months (2024-01, 2024-02)
- Revenue and units metrics

## Brand Compliance

The generated dashboard adheres to Kearney Design System (KDS) rules:
- Primary color: #7823DC (Kearney Purple)
- No green colors (forbidden)
- Dark mode background (#1E1E1E)
- No gridlines on charts
- Inter font family
- Data labels outside bars
